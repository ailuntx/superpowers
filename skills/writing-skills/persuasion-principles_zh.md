# 技能设计中的说服原则

## 概述

大语言模型（LLM）与人类对相同的说服原则产生反应。理解这种心理学有助于你设计更有效的技能——目的不是操控，而是确保即使在压力下也能遵循关键实践。

**研究基础：** Meincke 等人（2025 年）在 N=28,000 次 AI 对话中测试了 7 项说服原则。说服技巧使遵从率提高了一倍以上（33% → 72%，p < .001）。

## 七项原则

### 1. 权威性
**是什么：** 对专业知识、资历或官方来源的遵从。

**在技能中如何运作：**
- 命令式语言："你必须"、"绝不"、"总是"
- 不容商量的框架："没有例外"
- 消除决策疲劳和合理化借口

**何时使用：**
- 强化纪律的技能（TDD、验证要求）
- 安全关键实践
- 已确立的最佳实践

**示例：**
```markdown
✅ 先写代码再写测试？删掉它。重新开始。没有例外。
❌ 在可行时考虑先写测试。
```

### 2. 承诺一致性
**是什么：** 与先前行动、声明或公开表态保持一致。

**在技能中如何运作：**
- 要求宣告："宣告技能使用"
- 强制明确选择："选择 A、B 或 C"
- 使用追踪工具：用 TodoWrite 记录清单

**何时使用：**
- 确保技能被实际遵循
- 多步骤流程
- 问责机制

**示例：**
```markdown
✅ 当你找到一个技能时，你必须宣告："我正在使用 [技能名称]"
❌ 考虑让你的伙伴知道你正在使用哪个技能。
```

### 3. 稀缺性
**是什么：** 由时间限制或有限可用性产生的紧迫感。

**在技能中如何运作：**
- 有时限的要求："在继续之前"
- 顺序依赖："在 X 之后立即"
- 防止拖延

**何时使用：**
- 即时验证要求
- 时间敏感的工作流
- 防止"我稍后再做"

**示例：**
```markdown
✅ 完成任务后，在继续之前立即请求代码审查。
❌ 你可以在方便时审查代码。
```

### 4. 社会认同
**是什么：** 遵从他人的做法或被认为是常态的行为。

**在技能中如何运作：**
- 普遍模式："每次"、"总是"
- 失败模式："没有 Y 的 X = 失败"
- 建立规范

**何时使用：**
- 记录普遍实践
- 警告常见失败
- 强化标准

**示例：**
```markdown
✅ 没有 TodoWrite 追踪的清单 = 步骤会被跳过。每次都是。
❌ 有些人发现 TodoWrite 对清单有帮助。
```

### 5. 统一性
**是什么：** 共享身份、"我们感"、群体归属感。

**在技能中如何运作：**
- 协作性语言："我们的代码库"、"我们是同事"
- 共同目标："我们都想要质量"

**何时使用：**
- 协作工作流
- 建立团队文化
- 非等级化实践

**示例：**
```markdown
✅ 我们是共同工作的同事。我需要你诚实的技术判断。
❌ 如果我错了，你或许应该告诉我。
```

### 6. 互惠性
**是什么：** 回报所受恩惠的义务感。

**如何运作：**
- 谨慎使用——可能感觉像操控
- 技能中很少需要

**何时避免：**
- 几乎总是（其他原则更有效）

### 7. 喜好
**是什么：** 倾向于与我们喜欢的人合作。

**如何运作：**
- **不要用于确保遵从**
- 与诚实反馈文化冲突
- 产生阿谀奉承

**何时避免：**
- 在纪律强化方面总是避免

## 按技能类型组合原则

| 技能类型 | 使用 | 避免 |
|------------|-----|-------|
| 纪律强化型 | 权威性 + 承诺一致性 + 社会认同 | 喜好、互惠性 |
| 指导/技术型 | 适度权威性 + 统一性 | 强权威性 |
| 协作型 | 统一性 + 承诺一致性 | 权威性、喜好 |
| 参考型 | 仅需清晰性 | 所有说服原则 |

## 为何有效：心理学原理

**明确规则减少合理化：**
- "你必须"消除了决策疲劳
- 绝对语言消除了"这是例外吗？"的疑问
- 明确的反合理化设计堵住了特定漏洞

**执行意图创造自动行为：**
- 清晰触发条件 + 必要行动 = 自动执行
- "当 X 时，做 Y"比"通常做 Y"更有效
- 减少了遵从的认知负荷

**LLM 是类人的：**
- 在包含这些模式的人类文本上训练
- 训练数据中，权威性语言先于遵从行为
- 承诺序列（声明 → 行动）被频繁建模
- 社会认同模式（每个人都做 X）建立了规范

## 道德使用

**正当的：**
- 确保关键实践被遵循
- 创建有效的文档
- 防止可预见的失败

**不正当的：**
- 为个人利益而操控
- 制造虚假紧迫感
- 基于内疚的遵从

**检验标准：** 如果用户完全理解，这项技巧是否服务于他们的真实利益？

## 研究引用

**Cialdini, R. B. (2021).** *影响力：说服心理学（新版及扩展版）。* Harper Business.
- 七项说服原则
- 影响力研究的实证基础

**Meincke, L., Shapiro, D., Duckworth, A. L., Mollick, E., Mollick, L., & Cialdini, R. (2025).** 叫我混蛋：说服 AI 遵从令人反感的请求。宾夕法尼亚大学。
- 在 N=28,000 次 LLM 对话中测试了 7 项原则
- 使用说服技巧后遵从率从 33% 提高到 72%
- 权威性、承诺一致性、稀缺性最有效
- 验证了 LLM 行为的类人模型

## 快速参考

设计技能时，请自问：

1.  **它是什么类型？**（纪律型 vs. 指导型 vs. 参考型）
2.  **我试图改变什么行为？**
3.  **哪些原则适用？**（对于纪律型，通常是权威性 + 承诺一致性）
4.  **我组合的原则是否过多？**（不要七项全用）
5.  **这道德吗？**（服务于用户的真实利益吗？）
